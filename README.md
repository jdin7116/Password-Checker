# 🔐 Python Password Strength Checker 🔐
A simple Python program that evaluates the password strength of user inputs and provides real-time, color-coded feedback. The program guides users to create strong passwords by checking for length, letter case, numbers, and special characters, and visualizes strength with a bar.

## Features
1. Checks the user's inputted password for standard requirements:
* A minimum of 14 characters
* At least one uppercase and one lowercase letter
* At least one numerical digit
* At least one special character (i.e `@`, `$`, `!`, `%`, `*`, `?`, `&`)
2. Colour-coded feedback provided:
* Red: very weak
* Black: weak
* Yellow: moderate
* Blue: strong
* Green: very strong
3. Visual strength bar
4. Actionable tips for improvement
